# Сервис назначения ревьюеров для Pull Request'ов

Решение тестового задания осеннего набора на стажировку в Авито.

**Автор:** Чернышков Максим

## Запуск

```bash
docker compose up
```

Сервис будет запущен на порту 8080.

## Переменные окружения

Для настройки окружения нужно создать `.env` по аналогии с `.env.example` и поменять значения.

## Тестирование

```bash
go test ./...
```

Интеграционные тесты используют testcontainers. Можно запускать отдельно unit и интеграционные тесты:

```bash
go test ./internal/... # unit
go test ./integration/... # integration
```

## Дополнительные задания

- Реализована статистика (`/pullRequest/stats`) с подсчетом назначений по ревьюерам
- Интеграционные тесты для репозиториев и HTTP (testcontainers + httptest)
